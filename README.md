# Finanzas_personales_Dashboard
Dashboard de finanzas personales desarrollado en Power BI


## Descripción

Dashboard desarrollado en Power BI para analizar ingresos, gastos y capacidad de ahorro.

## Objetivos

- Analizar hábitos de consumo
- Identificar categorías de gasto
- Medir ahorro mensual
- Visualizar tendencias financieras

## Herramientas

- Power BI
- Power Query
- DAX
- Excel

## Funcionalidades

- KPIs financieros
- Evolución temporal
- Análisis por categoría
- Categorización automática de movimientos
